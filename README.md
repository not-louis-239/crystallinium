# Crystallinium

This is an aggregated library of all the Python stuff I keep making over and over. Maybe it will help someone out there as well.

I mostly made this because I'm tired of writing `def draw_text` a hundred bajillion times.

## Requirements

- Python 3.x (tested on 3.14.0, older versions may or may not work)
- To install Python package dependencies, run: `pip install -r requirements.txt`
